Free Code Camp Zipline: Get Weather App

This weather app has been created as per specification for Free Code Camp. To change the temperature between Celsius and Fahrenheit please click on the current temperature in the Bottom Left hand corner of the screen. At the moment this app determines the user's location via IP Address however, In future updates this will be changed to Co-ordinates. Please review the humans.txt file for more information about the developer, technologies & resources used in this project. This entire project file can be found on Github.
